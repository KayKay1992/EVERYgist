STEP 1: CREATE RELATEDPOSTS COMPONENT
--------------------------------------
Location: frontend/gist_hub/src/pages/Blog/components/RelatedPosts.jsx

Key Features:
‚úÖ Fetches posts with matching tags from backend API
‚úÖ Filters out the current post from results
‚úÖ Displays up to 3 related articles in a responsive grid
‚úÖ Beautiful card design with hover effects
‚úÖ Auto-scrolls to top when clicking related post

Component Props:
- currentPostId: ID of the current blog post (to exclude from results)
- tags: Array of tags from current post (for matching)

Logic Flow:
1. useEffect hook triggers when component mounts or props change
2. Takes first tag from current post's tags array
3. Calls API: GET /api/posts/tag/:tagName
4. Filters out current post by comparing _id
5. Limits results to first 3 posts (.slice(0, 3))
6. Displays posts in 3-column grid (1 column on mobile)

Special Handling:
- Loading state: Shows animated skeleton while fetching
- Empty state: Hides component if no related posts found
- Error handling: Gracefully logs errors, shows nothing on failure
- Scroll to top: window.scrollTo({ top: 0, behavior: 'smooth' })
