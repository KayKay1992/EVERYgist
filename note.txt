AI BLOG POST IDEAS SECTION - IMPLEMENTATION SUMMARY
===================================================

WHAT IT DOES:
A sidebar feature that automatically generates 5 creative blog post ideas using Google's Gemini AI when creating a new post. Each idea includes a title, description, tags, word count, and tone suggestion.

HOW IT WORKS:

1. BACKEND (aiController.js):
   - Created POST /api/ai/generate-ideas endpoint
   - Uses Google GenAI SDK with gemini-2.0-flash-lite model
   - Accepts topics, calls blogPostIdeasPrompt(), returns JSON array
   - Fixed API usage: ai.getGenerativeModel() → generateContent() → response.text()

2. PROMPT ENGINEERING (prompt.js):
   - blogPostIdeasPrompt() function creates structured AI prompt
   - Requests 5 ideas with: title, description, 3 tags, wordCount, tone
   - Returns valid JSON format

3. FRONTEND STATE (BlogPostEditor.jsx):
   - State: postIdeas, ideaLoading, openBlogPostGenForm
   - generatePostIdeas() function calls AI API with topics
   - useEffect auto-generates ideas on new post creation

4. UI COMPONENT (BlogPostIdeaCard.jsx):
   - Displays each idea as clickable card
   - Shows title with word count/tone badge
   - Description text and tag pills
   - Hover effects and onClick handler

5. LAYOUT:
   - 12-column grid: form (8 cols) + ideas sidebar (4 cols)
   - Only shows when creating new posts (!isEdit)
   - Responsive: stacks vertically on mobile
   - SkeletonLoader during API call

6. CONFIGURATION:
   - Environment: GOOGLE_API_KEY in .env.local
   - API Path: API_PATHS.AI.GENERATE_BLOG_POST_IDEAS
   - Authentication required

KEY BENEFITS:
✓ Instant creative suggestions to beat writer's block
✓ Structured ideas with SEO-friendly tags
✓ AI-powered content planning
✓ Seamless integration with blog editor
